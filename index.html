Quick Start.

Grab and build. 

	    $ git clone git@github.com:jhclark/ducttape.git
	    $ cd ducttape
	    $ ./build.sh
            # Add ducttape to your path
	    $ echo 'export PATH=$PWD:$PATH' >> ~/.bashrc
Do something. 

	    $ ducttape syntax/tutorial/1-basics/1-hello-world.tape
Now what? Try the tutorial.
